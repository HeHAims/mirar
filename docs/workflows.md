# MIRAR Workflows

## Daily
1. Morning check-in: sleep, dreams, body, emotion
2. Passive collection: smartwatch, phone activity
3. Active check-ins: 3 to 5 per day ("What is alive in you?")
4. Evening review: tag key nodes, set tomorrow's intention

## Weekly
- Emotional weather report
- Pattern recognition (loops, triggers)
- Growth edge summary

## Monthly
- Archetypal analysis: Hero's Journey, Rebirth, Cycles
- Season or moon alignment
- Narrative progression checkpoint
