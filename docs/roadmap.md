# MIRAR Roadmap

## Phase 1
- Text-only journal interface
- Manual node creation and connection
- Timeline visualization
- Outcome: awareness increase, dataset (~500 nodes)

## Phase 2
- Sentiment analysis (voice or text)
- Context auto-tagging (weather, GPS, time)
- Beta testing (10 users)
- Outcome: 5+ emotional patterns identified

## Phase 3
- Archetype matcher
- Intervention engine
- Artistic export (music or visualization)
- Outcome: proof of therapeutic and artistic application

## Phase 4
- Anonymous emotional "Wrapped" sharing
- Team or family resonance graphs
- Cultural resonance mapping
- Research API
- New emotional language based on data and story
